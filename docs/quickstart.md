Installation within virtualenv

If you don't have permission to install python packages, you can
install the requirements within virutalenv (requires pyton>=2.6):

> cd fbu
> virtualenv ENV
> source ENV/bin/activate
> pip -v install numpy
> pip -v install scipy
> pip -v install pylab
> pip -v install pymc
